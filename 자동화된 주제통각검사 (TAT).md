# Methodology: 자동화된 주제통각검사 (Automated TAT — Implicit Motive Elicitation)

> **예상 소요 시간:** 8~12분
>
> **세 줄 요약:**
> - 모호한 상황을 텍스트 장면으로 제시하고 "무슨 일이 일어나고 있으며 앞으로 어떻게 될 것 같은가, 인물은 무엇을 느끼고 무엇을 원하는가"를 자유 서술하게 한다.
> - 의식적 자기보고가 닿지 못하는 **암묵 동기(성취·친화·권력)** 를 투사를 통해 본인의 언어 그대로 포획한다.
> - 동기 코딩은 raw에 박지 않고 검증·활용 단계에서만 수행한다 — 원문은 불변으로 보존된다.
>
> **설계 핵심:**
> - **이론적 배경:** Murray의 주제통각검사(TAT), McClelland·Atkinson의 암묵 동기(nAch/nAff/nPow), Winter의 running-text 동기 채점
> - **동작 방식:** 표준화된 모호 장면 자극(텍스트)에 대한 자유 서술을 verbatim 수집. LLM은 추출 단계에 개입하지 않으며, 동기 코딩은 사후 분리 단계의 일이다.

## 0. 이론적 전제 및 설계 원칙

- **암묵 동기 ≠ 자기귀인 동기 (espoused–revealed 갭의 고전적 축):** McClelland(1989)는 투사 검사로 측정되는 **암묵 동기**가 설문으로 측정되는 **자기귀인(self-attributed) 동기**와 *체계적으로 다르며*, 장기적·자발적 행동을 예측하는 것은 전자라고 보였다. 이는 "공정성이 최우선"이라 쓴 사람이 독재자 게임에서 30%만 주는 본 프로젝트의 핵심 질문([README](<README.md>))과 정확히 같은 갭이다. TAT는 그 갭의 *동기* 측면을 전담한다.
- **투사 표면으로서의 모호성:** 자극은 *정답이 없는* 모호한 장면이다. 사용자는 빈 곳을 자신의 동기·우려·관계 모델로 채운다. 자극은 **선택 메뉴가 아니라 투사 표면**이므로, 무오염 원칙(설계자가 만든 객관식 배제, [Extracting the human mind](<Extracting the human mind.md>) §1-0)과 충돌하지 않는다 — 프레임을 주는 것이 아니라 *빈 프레임을 주고 사용자가 채우게* 한다.
- **무라벨 보존 (코딩과 저장의 분리):** nAch/nAff/nPow 같은 동기 코드를 **raw_store에 절대 박지 않는다.** 코드를 박는 순간 Winter 채점 체계의 편향이 영구 주입되고 미래 모델에 대해 진부화된다. 원문 서술만 저장하고, 동기 코딩은 전적으로 검증·활용 단계에서 동적으로 생성한다(이 프로젝트의 "사후 가공 불가" 원칙).
- **텍스트 자극 채택 (이미지 배제):** 저작권 있는 Murray 원판 카드 대신 **표준화된 텍스트 장면**을 쓴다. (a) 웹 환경 정합·접근성, (b) 자극 자체를 verbatim 로그에 남겨 자극–응답 쌍을 온전히 보존, (c) 자극을 균질하게 통제해 교차 인물 비교(라인업·X-시리즈) 가능.

## 1. 표준 자극 세트 (Standardized Stimulus Deck)

도메인(일/관계/자기)별로 **동기 3축이 모두 활성화될 수 있도록** 양가적·모호한 장면을 사전 설계한다. 각 장면은 *어느 동기로도 읽힐 수 있게* 의도적으로 비결정적이다(특정 동기를 유도하지 않는다).

### 도메인 A 일/과업

1. `(성취/권력 양가)` "한 사람이 늦은 밤 사무실에 혼자 남아 화면을 보고 있다. 책상에는 막 도착한 한 통의 메일이 열려 있다." 〔투사 표면: 야망/평가/통제〕
2. `(친화/권력 양가)` "회의가 끝나고 사람들이 나가는데, 두 사람이 남아 마주 보고 있다. 한 명은 서류를 든 채 멈칫한다." 〔투사 표면: 위계/인정/대립〕

### 도메인 B 관계/소통

3. `(친화/상실 양가)` "두 사람이 마주 보고 있는데, 한 명은 고개를 숙이고 있다." 〔투사 표면: 거절/돌봄/거리〕
4. `(친화/자율 양가)` "한 사람이 문 앞에 서 있고, 안쪽에서 다른 사람이 등을 보이고 있다." 〔투사 표면: 접근/회피/경계〕

### 도메인 C 자기/라이프

5. `(성취/두려움 양가)` "한 사람이 갈림길 앞에 서 있다. 한쪽 길은 밝고 한쪽은 흐릿하다." 〔투사 표면: 선택/정체/두려운 자기〕
6. `(권력/무력 양가)` "큰 방 한가운데에 한 사람이 서 있고, 주변에 빈 의자들이 놓여 있다." 〔투사 표면: 지배/고립/지위〕

### 표준 발문 (Fixed Probe)

각 장면에 대해 **고정된 4구성 서술**을 요청한다(이 4요소가 Winter/McClelland 채점의 입력 면을 모두 연다):

1. 지금 무슨 일이 일어나고 있는가?
2. 어쩌다 이 상황이 되었는가? (직전 사건)
3. 인물(들)은 무엇을 **느끼고**, 무엇을 **원하는가**?
4. 앞으로 어떻게 될 것 같은가? (결말)

## 2. 추출 실행 (Execution)

- **LLM 제로 개입:** [생성 은유와 문장 완성](<생성 은유와 문장 완성.md>) §2와 동일하게, 추출 중 LLM이 모호성 지적·추가 질의를 일절 하지 않는다. 서술을 강제하지 않으며, UI 단의 `minlength` 제약은 우발적 제출을 막는 최소치(예: 5~10자)로만 설정하여 짧은 응답이나 침묵도 사용자의 고유한 데이터로 존중한다. 가이드는 "상황을 자유롭게 이어서 적어주세요"와 같이 구조적으로만 제시한다.
- **순서·피로 통제:** Core Heavy에 준하는 인지 부하이므로([Extracting the human mind](<Extracting the human mind.md>) §1-0-5), 도메인당 1~2장면으로 세션을 쪼개고 삼항 도출과 같은 세션에 몰지 않는다.
- **가명화 시 verbatim 보존:** 인명/고유명사만 `[Name_A]`로 치환하고 어투·구조는 한 글자도 바꾸지 않는다(생성 은유 §2와 동일 원칙).

## 3. 산출물 및 데이터 저장 (Raw Store Append)

자극(prompt)과 사용자 원문(verbatim)을 함께 저장한다. **동기 코드 필드는 두지 않는다.**

```yaml
  - id: tat0003
    ts: 2026-06-21T22:10:00+09:00
    method: tat
    domain: relation
    stimulus_id: B-3                      # 표준 자극 식별자 (교차 인물 비교용)
    prompt: "두 사람이 마주 보고 있는데, 한 명은 고개를 숙이고 있다."
    raw_verbatim: >
      고개 숙인 쪽이 뭔가 잘못해서 사과하는 중인 것 같다. 근데 사과를 받는
      쪽이 더 불안해 보인다. 결국 받아주긴 할 텐데, 마음속으로는 또 이런 일이
      반복될까 봐 지친 상태일 것 같다. 앞으로 둘은 겉으론 화해해도 거리가 생긴다.
    # 동기 코드(nAch/nAff/nPow)는 여기에 저장하지 않는다 — 검증·활용에서 동적 생성
```

## 4. 알려진 한계 및 활용 원칙 (Limitations)

- **고전 TAT의 낮은 심리측정 신뢰도 (정면 인정):** 전통적 TAT는 채점자 간 신뢰도·재검사 신뢰도가 약하다는 강한 비판을 받아 왔다. 본 모듈은 이를 두 가지로 방어한다 — (a) **face validity가 아니라 예측 타당도로만 평가**한다(서술이 "그럴듯한가"가 아니라 *실제 행동을 맞히는가*, [검증 모드](<검증 모드.md>)), (b) 단일 장면이 아니라 **복수 장면 집계**로 개인 동기 추정의 신뢰구간을 좁힌다(장면당 노이즈는 누적으로 상쇄).
- **자극 균질성의 양면:** 표준 자극은 교차 인물 비교를 가능케 하지만(라인업 미끼·X-시리즈), 동시에 모두에게 같은 자극이라 *생태학적 맥락*은 약하다. 따라서 TAT는 ESM·CCRT 같은 맥락 강한 메서드와 **교차검증**으로만 해석하고 단독 판정하지 않는다.
- **안전 필터 우회:** 투사 서술에 폭력·자기파괴적 내용이 나올 수 있다. 활용 단계 프롬프트 최상단에 임상적 목적 고지를 넣어 상용 LLM의 순화(sanitization)를 차단한다(생성 은유 §4-3과 동일).
- **검증으로 가는 길 (Part 2):** 암묵 동기 추정의 *증분 타당도*를 검증한다 — "TAT 서술에서 추정한 암묵 동기"가 자기보고 가치(가치 할당·래더링)를 *넘어서* 실제 행동(CCRT 반응·ESM)을 예측하는지. 이것이 McClelland의 암묵–자기귀인 동기 분리를 LLM으로 자동 재현하는 핵심 실험이다([검증 모드](<검증 모드.md>) A-07).

## 5. UI 렌더링 명세 (Frontend Component)
- **UI 컴포넌트:** `장면 카드 + 단계형 자유 서술(Stepped Free-text)`
- **상호작용:**
  - 모호한 텍스트 장면을 카드 상단에 고정 노출하고, 아래 4개 발문을 한 번에 하나씩(stepped) 제시해 서술을 받는다.
  - 입력 완료 후 다음 발문으로 넘어가면 이전 서술은 수정 불가(인상 관리 차단, 생성 은유 §2 비가역성과 동일).
